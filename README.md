## Design notes
Product chosen: luxury watch (Casio Edifice, Patek Philippe, Seiko Astron)
References: Godly.website, Patek Philippe official press room
Signature effect recreated: Magic UI border-glow cards, moving aurora liquid color profiles, and animating SVG gear train visualizers - built with AI assistance.

## Stack
HTML5 · CSS3 · AI-assisted animations (Antigravity Code Assistant) · GitHub Pages

## 🤖 AI usage · 📚 What I learned
I programmed the layouts and typography spacing. AI assisted in writing coordinates tracking code for the spotlight hover border glow cards, structured shifting CSS variables for the color-morphing theme selectors, and coded rotation speeds of the SVG complication gears.
